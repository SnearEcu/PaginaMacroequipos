🎨 Guía de Personalización - Ejemplos Prácticos

Aquí tienes ejemplos de cómo personalizar tu página web con información real:

1. Cambiar el Nombre de la Empresa

Buscar en index.html:

<title>Tu Empresa - Soluciones Industriales</title> <h2>TuEmpresa</h2> <p>info@tuempresa.com</p>

Cambiar por:

<title>Industrias García - Soluciones Industriales</title> <h2>Industrias García</h2> <p>contacto@industriasgarcia.com</p>

2. Personalizar Servicios

Ejemplo para una empresa de metalurgia:

<div class="service-card"> <div class="service-icon"> <i class="fas fa-industry"></i> </div> <h3>Fundición de Metales</h3> <p>Especialistas en fundición de hierro, acero y aluminio para piezas industriales de alta calidad.</p> </div> <div class="service-card"> <div class="service-icon"> <i class="fas fa-cogs"></i> </div> <h3>Mecanizado CNC</h3> <p>Mecanizado de precisión con tecnología CNC para piezas complejas y prototipos.</p> </div>

3. Personalizar Productos

Ejemplo para una empresa de maquinaria:

<div class="product-card"> <div class="product-image"> <i class="fas fa-industry"></i> </div> <h3>Prensas Hidráulicas</h3> <p>Prensas hidráulicas de 50 a 500 toneladas para conformado de metales y plásticos.</p> </div> <div class="product-card"> <div class="product-image"> <i class="fas fa-conveyor-belt"></i> </div> <h3>Líneas de Ensamblaje</h3> <p>Líneas automatizadas de ensamblaje para producción en serie con control de calidad integrado.</p> </div>

<div class="contact-item"> <i class="fas fa-map-marker-alt"></i> <div> <h4>Dirección</h4> <p>Av. 5. Personalizar Colores

En styles.css, cambiar los colores principales:

/* Para una empresa de construcción (colores tierra) */ .nav-logo h2 { color: #8B4513; } .btn-primary { background: #8B4513; } .service-icon { background: linear-gradient(135deg, #8B4513, #A0522D); } /* Para una empresa tecnológica (colores azul/verde) */ .nav-logo h2 { color: #00BFFF; } .btn-primary { background: #00BFFF; } .service-icon { background: linear-gradient(135deg, #00BFFF, #32CD32); } /* Para una empresa de energía (colores naranja) */ .nav-logo h2 { color: #FF8C00; } .btn-primary { background: #FF8C00; } .service-icon { background: linear-gradient(135deg, #FF8C00, #FFA500); }

6. Agregar Imágenes Reales

Crear carpeta images/ y agregar imágenes:

<!-- En lugar del placeholder --> <div class="hero-placeholder"> <i class="fas fa-industry"></i> </div> <!-- Usar imagen real --> <div class="hero-image"> <img src="images/fabrica-principal.jpg" alt="Nuestra fábrica"> </div>

Y agregar estos estilos en CSS:

.hero-image img { width: 100%; max-width: 400px; height: auto; border-radius: 20px; box-shadow: 0 20px 40px rgba(0,0,0,0.3); }

7. Personalizar SEO

Agregar en el <head> de index.html:

<meta name="description" content="Industrias García - Líder en soluciones industriales y maquinaria pesada. Especialistas en fundición, mecanizado CNC y automatización industrial en Monterrey."> <meta name="keywords" content="industria, maquinaria, fundición, CNC, automatización, Monterrey, México"> <meta property="og:title" content="Industrias García - Soluciones Industriales"> <meta property="og:description" content="Especialistas en maquinaria industrial y automatización"> <meta property="og:image" content="https://tuempresa.com/images/logo.png"> <meta property="og:url" content="https://tuempresa.com">

8. Configurar Formulario de Contacto

Paso 1: Ir a formspree.io

Paso 2: Crear cuenta y formulario

Paso 3: Reemplazar en index.html:

<form action="https://formspree.io/f/xaybzwkd" method="POST">

Nota: Reemplaza "xaybzwkd" con tu endpoint real de Formspree

🎯 Consejos de Personalización

  • Mantén consistencia: Usa los mismos colores y fuentes en toda la página
  • Imágenes de calidad: Usa imágenes profesionales de tu empresa
  • Contenido relevante: Describe servicios y productos específicos de tu negocio
  • Información de contacto: Asegúrate de que sea fácil contactarte
  • Testimonios: Agrega opiniones de clientes satisfechos
  • Certificaciones: Muestra certificaciones y premios de tu empresa
